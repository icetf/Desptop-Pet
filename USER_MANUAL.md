# 桌面健康助手 (DesktopPet) 用户手册

## 1. 产品简介
桌面健康助手是一款结合了桌面宠物、智能健康提醒、待办事项管理和内容聚合功能的综合性效率工具。它致力于在提升您工作效率的同时，保护您的身体健康。

## 2. 安装与卸载
- **安装**: 
  - Windows: 运行 `DesktopPet-Setup.exe`，按提示操作即可。支持选择安装目录，自动创建桌面快捷方式。
  - macOS: 打开 `DesktopPet.dmg`，将应用拖入 Applications 文件夹。
  - 便携版: 解压 `DesktopPet-portable.zip` 直接运行主程序。
- **卸载**:
  - Windows: 在控制面板中找到“DesktopPet”，点击卸载。卸载程序会自动清理注册表和本地数据。
  - macOS: 将应用移至废纸篓。

## 3. 核心功能说明

### 3.1 智能休息提醒系统
- **定时设置**: 在“系统设置”面板中，支持自定义 5-120 分钟的休息间隔。
- **交互方式**:
  - 当触发提醒时，宠物会弹出气泡提示。
  - 若用户持续操作键盘鼠标（活动检测），系统将执行**强制黑屏保护**，显示全屏休息界面，倒计时结束前无法关闭。
  - 可选择“稍后提醒”（延迟 15 分钟）或“立即休息”。

### 3.2 桌面宠物交互
- **悬浮窗体**: 宠物支持全屏透明显示，按住宠物可自由拖拽。
- **外观自定义**: 支持 PNG/JPG (≤5MB) 或 GIF/MP4 (≤10MB)。
- **互动**: 左键点击宠物可触发对话气泡和放大动画。

### 3.3 智能内容服务
- 打开控制面板切换至“智能内容”，可查看：
  - **实时天气**: 根据当前 IP 自动获取。
  - **每日笑话**: 随机生成，放松心情。
  - **热点新闻**: 聚合主流新闻，支持分类。

### 3.4 待办事项管理
- 基于**四象限法则**（重要紧急、重要不紧急、不重要紧急、不重要不紧急）进行任务分类。
- 支持完成状态切换、任务删除等操作，数据加密存储于本地。

## 4. 故障排除
- **Q: 宠物窗口显示黑底而不是透明？**
  - A: 请检查系统显卡驱动是否正常，或在设置中开启/关闭硬件加速。
- **Q: 无法开机自启？**
  - A: 请检查杀毒软件（如 360、Windows Defender）是否拦截了自启项。本软件承诺无恶意代码，请添加白名单。
- **Q: 数据丢失？**
  - A: 数据加密存储在本地，请勿手动修改。数据均采用 AES-256 加密，无法直接读取。

## 5. 隐私声明
本应用所有数据（如待办事项、偏好设置）均采用 **AES-256 加密**并保存在本地 `electron-store` 存储中，不上传任何云端服务器，保障您的隐私安全。
