# DesktopPet

一个基于 Electron 的桌面宠物与健康提醒应用，集成了桌宠互动、休息提醒、待办事项管理、天气查询、每日笑话和控制面板等功能。

## 项目特点

- 透明悬浮桌宠，支持拖拽移动、边缘吸附、右键菜单
- 休息提醒与黑屏休息页联动，帮助降低久坐和持续用眼压力
- 四象限待办事项管理，支持提醒时间、到点提示、完成切换与删除
- 控制面板支持手动设置天气城市
- 支持切换内置桌宠风格，并可上传自定义图片作为桌宠样式
- 自定义日期时间选择器与统一风格弹窗

## 功能概览

### 桌面宠物

- 左键点击宠物可触发互动气泡
- 支持自由拖拽到桌面任意位置
- 右键可打开设置、切换置顶、隐藏宠物、退出程序
- 支持上传 PNG / JPG / JPEG / GIF / WEBP / SVG 作为桌宠图片

### 健康提醒

- 可设置休息提醒间隔
- 到达提醒时间后，桌宠会显示提醒气泡
- 支持触发黑屏休息界面

### 待办事项

- 按四象限优先级分类
- 支持设置提醒日期与时间
- 到点后触发桌宠提醒和系统通知
- 待办数据保存在本地

### 智能内容

- 天气：支持自动识别或手动设置城市
- 每日笑话：联网随机获取

### 控制面板

- 待办事项面板
- 智能内容面板
- 系统设置面板
- 自定义弹框与自定义时间选择器

## 本地运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发版本

```bash
npm start
```

## 打包

### Windows

```bash
npm run build:win
```

### macOS

```bash
npm run build:mac
```

## 数据存储

- 设置与待办数据使用本地存储保存
- 用户上传的桌宠图片会复制到项目本地数据目录
- `.user-data/`、`node_modules/` 等已加入忽略列表，不会推送到仓库
